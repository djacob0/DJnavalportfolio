# Portfolio Website

## Built With

- JavaScript & CSS
- React JS
- Redux, Webpack, Babel and Jest 


## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites
- Node JS

### Setup
git clone 

### Install
npm install

### Usage
npm start

### Run tests
npm test

### Deployment
npm run build
